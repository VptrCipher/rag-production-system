Explain the architecture of the current hybrid search retrieval system.

The system implements a Two-Stage Hybrid Retrieval pipeline:

  1. Stage 1: Parallel Sparse (BM25) and Dense (Cosine) retrieval.
  2. Stage 2: Reciprocal Rank Fusion (RRF) followed by a Cross-Encoder reranker.
def retrieve_hybrid(query):
    dense = qdrant.search(query_vector)
    sparse = bm25.search(query_text)
    return rerank(dense, sparse)
